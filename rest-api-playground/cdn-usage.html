The REST API Playground bundle is available through multiple CDN providers for easy embedding in your applications.

CDN Provider URL Pattern Features
JSDelivr Recommended https://cdn.jsdelivr.net/gh/rmarchet/rest-api-playground@VERSION/dist/bundle.js Version pinning, fast global CDN, reliable
GitHub Pages CDN Always Latest https://rmarchet.github.io/rest-api-playground-cdn/bundle.js Always latest version, direct from source
Unpkg https://unpkg.com/rest-api-playground@VERSION/dist/bundle.js NPM-based, version pinning available
GitHub Releases https://github.com/rmarchet/rest-api-playground/releases/download/VERSION/rest-api-playground.js Direct downloads, versioned assets

üé® Configuration Examples

Minimal Setup:

const playground = new RestAPIClient(); playground.init({ container: 'playground' });

Customized Configuration:

const playground = new RestAPIClient({ config: { title: 'Payment API', subtitle: 'Process payments securely', theme: 'dark', showHeader: true, hideRequestPanel: false, requestLabel: 'API Request', responseLabel: 'API Response', defaultUrl: 'https://api.example.com/payments' } }); playground.init({ container: 'payment-playground' });

Embedded in Documentation:

const playground = new RestAPIClient({ config: { showHeader: false, hideRequestPanel: true, responseLabel: 'Try it now ‚Üí', defaultUrl: 'https://api.docs.com/users/me' } }); playground.init({ container: 'inline-demo' });
