üéØ Theme Configuration Examples

Basic Theme Setup:

// Light theme (default)
const lightPlayground = new RestAPIClient({
  config: {
    theme: 'light',
    title: 'Light Theme API Tester'
  }
});

// Dark theme
const darkPlayground = new RestAPIClient({
  config: {
    theme: 'dark',
    title: 'Dark Theme API Tester'
  }
});

Dynamic Theme Switching:

// Initialize with light theme
const playground = new RestAPIClient({
  config: { theme: 'light' }
});

// Switch to dark theme
playground.updateConfig({ theme: 'dark' });

// Switch back to light
playground.updateConfig({ theme: 'light' });

Combined with Other Options:

const playground = new RestAPIClient({
  config: {
    theme: 'dark',
    hideRequestPanel: true,
    requestLabel: 'Query',
    responseLabel: 'Data',
    title: 'Dark GraphQL Explorer'
  }
});

üíª Technical Implementation

CSS Variables Definition:

/* globals.css */
:root {
  /* Light theme variables */
  --color-bg-primary: #ffffff;
  --color-text-primary: #1e293b;
  /* ... more variables */
}

.theme-dark {
  /* Dark theme overrides */
  --color-bg-primary: #1f2937;
  --color-text-primary: #f9fafb;
  /* ... more overrides */
}

Component Usage:

/* Component CSS */
.panel {
  background: var(--color-bg-primary);
  color: var(--color-text-primary);
  border: 1px solid var(--color-border-primary);
  box-shadow: var(--shadow-md);
}

Theme Class Application:

// App.tsx
return (
  <div className={`app theme-${theme}`}>
    {/* All child components inherit theme */}
  </div>
);
